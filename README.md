# TEKIYOMI#2

日本語テキストを音声で読み上げるWebアプリです。スマートフォン（iPhone）での使用に最適化されています。

## 機能

- テキストを入力して再生・一時停止・停止ができる
- 読み上げ中のテキストをハイライト表示
- 再生速度を変更できる（1x / 1.5x / 2x）
- シークバーで再生位置を自由に移動できる
- 15秒巻き戻しボタン
- 入力したテキストの所要時間（読み上げ時間）をリアルタイム表示
- ピンチズーム無効化などiOS向けの最適化済み

## 使い方

1. テキストエリアに読み上げたいテキストを貼り付ける
2. 再生ボタンを押す
3. 一時停止・停止・巻き戻しで操作する

## 動作環境

ブラウザのWeb Speech API（`speechSynthesis`）を使用しています。Safari / Chrome で動作します。音声には日本語音声（Kyoko）を使用します。

## 起動方法

`tts-japanese.html` をブラウザで直接開くだけで動作します。サーバー不要です。
